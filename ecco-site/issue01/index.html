How To Steal Photoshop For Life

Step 1: Download Adobe Creative Cloud Desktop App

It might sound counter-intuitive, but you will need the real version of Adobe apps in order to get them for free. No need to go to any shady websites to get a pre-cracked version. You can download it here: Adobe's Website

Step 2: Download Adobe Zii

Adobe Zii is a wonderful program that makes the Adobe 7 day trial period last forever! It is totally safe to use, does not need internet access to run and is under 15MB in size. You can download it here: MacApps.to

Step 3: Download LuLa

LuLa is a free program that prevents other programs from accessing the internet. Adobe Zii is a great tool, but to be sure you get to keep Photoshop for life, download LuLa here: LuLa Homepage

LuLa works on a 'rule' system: either block or allow. You will need to manually set rules to block all of the programs listed in the screenshot.

Step 4: Start Photoshop Trial

Like I said before, you are going to use real Adobe software! Click the "try" button in the Creative Cloud Desktop App that you downloaded earlier.

You might need to sign in before you download. That is 100% okay! Feel free to sign in with a real Adobe Account.

Step 5: Run Adobe Zii

After you have fully installed Photoshop, try not to open it yet. It's okay if you do, but it will be faster this way. Navigate to the Adobe Zii program that you downloaded earlier.

There should be only 1 button in the program window, labeled "PATCH OR DRAG". Click that button. You will be prompted to enter your password. This is just so Adobe Zii can go and mess around with the Adobe files deep in your computer. If everything worked out, you should see green text in Adobe Zii that reads "COMPLETED".
